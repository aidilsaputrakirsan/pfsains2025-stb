Sistem Monitoring Kualitas Pangan Multi-Komoditas Berbasis IoT

Untuk Ketahanan Pangan UMKM di Koridor IKN Kalimantan Timur: Platform Rantai Pasok B2B dengan Jaminan Kualitas Waktu Nyata

🎯 Visi & Tujuan Proyek

🌟 Visi Utama

Mengembangkan sistem monitoring kualitas pangan berbasis IoT yang terintegrasi dengan platform digital B2B untuk meningkatkan ketahanan pangan UMKM di koridor IKN Kalimantan Timur.

💡 Inovasi Kunci

Sensor array IoT multi-parameter (suhu, kelembaban, pH, gas volatil) dengan Raspberry Pi untuk monitoring real-time tiga komoditas: ikan segar, pisang cavendish, dan kelapa muda.

🎯 Target Dampak

Mengurangi post-harvest loss 20-25%, meningkatkan pendapatan UMKM 30-40%, dan memperkuat ketahanan pangan regional untuk mendukung pembangunan IKN.

⚙️ Cara Kerja Sistem IoT

1Data Acquisition

Sensor array mengumpulkan data suhu, kelembaban, pH, gas volatil setiap 5 menit → Raspberry Pi processing → QR code batch tracking

2Cloud Analytics

Data real-time ke cloud platform via 4G/WiFi → Machine learning prediction → Alert system untuk deteriorasi kualitas

3B2B Platform

Dashboard UMKM untuk monitoring → Buyer app untuk quality verification → Direct order dengan jaminan kualitas real-time

4Quality Assurance

Predictive analytics untuk shelf-life → Certification digital untuk BPOM compliance → Traceability end-to-end

🐟 Komoditas Target & Market Focus

🐟 Ikan Segar (40%)

Ikan kerapu, tongkol dari Balikpapan → Restoran/hotel IKN. Monitoring suhu (0-4°C), pH (6.0-6.8), ammonia detection untuk freshness.

🍌 Pisang Cavendish (35%)

Pisang kualitas ekspor PPU → Ritel modern IKN. Monitoring ripeness, ethylene concentration, firmness untuk optimal timing.

🥥 Kelapa Muda (25%)

Produk lokal khas Kaltim → Industri F&B IKN. Monitoring pH air kelapa (4.1-5.4), microbiological contamination detection.
