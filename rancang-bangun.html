üîÑ Alur Data dan Proses

1

Data Acquisition

Sensor array mengumpulkan data suhu, kelembaban, pH, gas volatil, berat, dan warna setiap 5 menit secara kontinyu dari produk pangan.

2

Local Processing

Raspberry Pi melakukan preprocessing, filtering, dan validasi data. Edge computing untuk response cepat dan local storage sebagai backup.

3

Data Transmission

Data dikirim ke cloud platform melalui 4G/WiFi menggunakan protokol MQTT dengan enkripsi SSL/TLS untuk keamanan.

4

Cloud Analytics

Machine learning models menganalisis data untuk prediksi kualitas, deteksi anomali, dan optimasi shelf-life produk.

5

User Notification

Alert real-time dikirim ke UMKM dan buyer melalui mobile app dengan rekomendasi tindakan untuk menjaga kualitas.
